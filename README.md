# LAN Setup and IP Addressing

## Project Overview
This project demonstrates the design and configuration of a Local Area Network (LAN) using Cisco Packet Tracer.

## Devices Used
- 1 Router
- 1 Switch
- 5 PCs
- 1 Printer

## Network Features
- IPv4 addressing
- Static and DHCP configuration
- Router as DHCP server
- Successful LAN communication

## Technologies
- Cisco Packet Tracer
- IPv4
- DHCP
- Static Routing
